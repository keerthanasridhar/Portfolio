COMPETITIVE ANALYSIS:
This project is broken down to:
Illustraion for the cafe

Exploring graphic design techniques

Creating an interactive application for the order menue

Creating a physical portfolio

Illustration where done with respect to products at the Flying Fruit Cafe.Print design starting from Napkin holder,breakfast order print menu,Print design for Airpot.This is what got my started with the graphic designing.

Idea Generation:

The cafe had plain order sheets to order and wanted an interactive application to order.So,visiting the cafe I figured that the interiors of the cafe where all with vibrant colors with whimsical writeups and so I followed the same for the application so that it could give the customer a familarity of using the application.

Physical Portfolio

I had to create a physical portfolio to showcase all the designs and from your own thought process.I created a color palette which could represnt all the designs that I created for the class.
